# Note-Prediction
Projet de machine learning pour prédire les notes des étudiants. Utilise des modèles avancés pour une précision élevée. Facile à utiliser et ouvert aux contributions.
## Prérequis

- Python
- Bibliothèques Python : pandas, numpy, seaborn, matplotlib, scikit-learn

## Installation

1. Clonez ce dépôt sur votre machine locale :

   ```bash
   git clone https://github.com/medrifai/Note-Prediction.git
Installez les dépendances requises :

bash
Copy code
pip install -r requirements.txt
Utilisation
Assurez-vous que vous avez Python et les bibliothèques nécessaires installés.

Exécutez le script Python prediction_note.py :

bash
Copy code
python prediction_note.py
Cela chargera les données à partir du fichier Excel, effectuera une détection d'outliers, supprimera les valeurs aberrantes, puis utilisera un modèle de régression pour prédire les notes en fonction de la participation en classe et du temps d'étude.

Consultez les résultats et visualisations générés.

Fichier de données
Les données utilisées pour ce projet sont stockées dans le fichier Excel PREDICTION NOTE.xlsx situé dans le répertoire data.

Résultats
Le script génère des résultats, y compris des statistiques descriptives, la détection d'outliers, des histogrammes, et des visualisations de relations entre les variables.

Modèle de Régression
Le modèle utilisé pour la régression est le Gradient Boosting Regressor de scikit-learn. Les performances du modèle sont évaluées à l'aide de la validation croisée.
